报错处理

  • 如果在使用过程中遇到 libusb_open() failed with libusb_error_not_supported 可以参考 https://stackoverflow.com/questions/41066653/openocd-error-libusb-open-failed-with-libusb-error-not-supported，把 contrib 目录下的 60-openocd.rules（不同版本可能会不一样）拷贝到 /etc/udev/rules.d
  • 在使用 gdb-multiarch 进行 stm32 调试的时候，pwndbg 会在 target remote 命令执行后挂起，详细可见 github 上的 `issue https://github.com/pwndbg/pwndbg/issues/264，解决方法就是换用其他插件或者是不用
