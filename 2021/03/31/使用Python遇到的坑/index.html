使用Python遇到的坑

 2021-03-31 2021-04-06 0次访问

现在基本上还是语法上遇到的坑（我 Python 学得不是很好…），在博客里记录一下

  • Python 的运算优先级与 c 不同
    例：异或运算符 ^ 的优先级比 + 号低

  • v2 = “字符串”，这里的 v2 是一个元组（不可改变元素值的列表），所以无法直接对单个元素赋值

  • python 文件名不能与 import 的模块名相同
